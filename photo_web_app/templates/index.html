ようこそ

このシステムでは、Google Photosに保存されている写真の中から、特定の子供が写っている写真を自動的に見つけることができます。

使い方

  1. Googleアカウントでログインし、Google Photosへのアクセスを許可します
  2. 対象となる子供の写真を3枚程度アップロードします
  3. システムが顔の特徴を学習します
  4. Google Photos内の写真をスキャンし、同じ子供が写っている写真を特定します
{% if authenticated %} 子供の写真をアップロード ログアウト {% else %} Googleでログイン {% endif %}

プライバシーとセキュリティ

  • アップロードされた写真は顔特徴の抽出のみに使用され、保存されません
  • Google Photosの写真は読み取り専用でアクセスされます
  • 顔認識データはセッション中のみ保持され、ログアウト時に削除されます
